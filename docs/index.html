Les marchés, en temps réel.
Sans une donnée inventée.

Monere est une plateforme de suivi des marchés : cotations US/EU en temps réel, earnings officiels, suivi du « smart money » et analyses IA — où chaque chiffre affiché est relié à sa source primaire (SEC, Congrès US, Finnhub, Yahoo Finance).

IA fiabilisée

Une IA qui cite ses sources — ou se tait

Aperçus simulés à des fins de démonstration — dans l'application, chaque donnée affichée est réelle et reliée à sa source.

💡 Pourquoi ce projet

Démontrer qu'on peut construire un produit financier grand public où la confiance est le cœur du design : aucune donnée inventée, chaque chiffre traçable jusqu'à sa source primaire — y compris dans les fonctions IA.

🖥️ Hébergement

Le serveur tourne sur le poste du créateur et est exposé via un tunnel sécurisé (coût : 0 €). Cette page — toujours en ligne — détecte sa disponibilité en direct et vous y emmène quand il est allumé.

🔍 Pour aller plus loin

Le dépôt GitHub contient le code complet, l'architecture détaillée (BRD), le guide de déploiement cloud et la politique de sécurité. Compte de test disponible sur demande.
